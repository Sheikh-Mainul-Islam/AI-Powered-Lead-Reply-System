# AI Lead Autoresponder System 

This project automatically replies to new coaching or consulting leads using AI.

When a new lead is added into Google Sheets, an AI-powered message is generated based on the lead status.

This system is designed to be:
✓ Simple
✓ Non-technical
✓ Easy for business owners to use

---

## What It Does

- Detects new lead entries
- Sends a friendly AI-generated reply
- Works when status = New or Follow-up
- Supports lowercase and uppercase inputs

Example statuses:
New
new
Follow-up
follow up

---

## Tools Used

- Google Sheets
- Zapier / Make (automation builder)
- Chat AI model

---

## Ideal Users

- Coaches
- Consultants
- Online experts
- Personal brands

---

## Why This Matters

Most leads never reply back simply because business owners reply late.

This system:

✓ replies instantly  
✓ keeps conversations alive  
✓ increases booked call rate  
✓ reduces stress
